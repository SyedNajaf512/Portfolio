# Syed Najaf Turab - Portfolio

A premium, high-tech portfolio website built with Next.js 14, Tailwind CSS, and Framer Motion.

## Features

*   **Modern UI/UX**: Dark theme, glassmorphism, and neon accents.
*   **Animations**: Smooth transitions powered by Framer Motion.
*   **Sections**: Hero, Services, Skills, Projects, and Contact.
*   **Contact Form**: Fully functional integration with EmailJS.
*   **Responsive**: Optimized for mobile, tablet, and desktop.

## Tech Stack

*   **Framework**: Next.js 14 (App Router)
*   **Styling**: Tailwind CSS
*   **Icons**: Lucide React
*   **Language**: TypeScript

## Getting Started

1.  **Install dependencies**:
    ```bash
    npm install
    ```

2.  **Set up Environment Variables**:
    Update `src/components/sections/Contact.tsx` with your EmailJS keys.

3.  **Run the development server**:
    ```bash
    npm run dev
    ```

4.  **Open the application**:
    Navigate to [http://localhost:3000](http://localhost:3000) in your browser.

## Deployment

Deploy easily on [Vercel](https://vercel.com) by connecting this repository.
